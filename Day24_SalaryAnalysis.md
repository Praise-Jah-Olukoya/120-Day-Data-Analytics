## Day 24/120 — Project 2: Salary Analysis 

Project 2 of the salary analysis is wrapped up. It was a full session but the results made it worth it.

The project answered four questions:

- Do more skills equal more money?
- What are the top skills employers are looking for?
- Which skills pay the most?
- How do US salaries compare to Non-US salaries?

To answer these questions I worked with Power Pivot and DAX — building measures using COUNT, DISTINCTCOUNT, MEDIAN, DIVIDE and CALCULATE. 
Each function had a specific job and putting them all together to answer real questions made the session feel worthwhile.

The project is done and I'm proud of that.

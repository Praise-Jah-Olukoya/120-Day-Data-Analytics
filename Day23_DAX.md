Day 23/120 — DAX

Today I expanded my understanding of DAX by exploring how calculations behave inside a data model.

I learned about measures and calculated columns. Measures are dynamic calculations used for report-level summaries, while calculated columns are stored in the table when data is loaded.

I also calculated median salary using the MEDIAN function and explored CALCULATE, which helps modify filter context so calculations can be performed for specific groups.

Big takeaway:
DAX is not just about writing formulas. It is about understanding how data is calculated, filtered, and structured inside a model.
